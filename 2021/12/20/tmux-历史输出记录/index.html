如何优雅使用 tmux

发表于 2021-12-20 | 更新于 2021-12-25
总字数: 656 | 阅读时长: 2分钟

tmux 的使用
为啥要用 tmux 最大的好处是可以看会话中的历史输出
以及关闭窗口会, 程序会继续执行

所有窗口
tmux ls

新建一个会话
tmux new -s bb

进入会话
tmux a -t 编号或名字
(a的意思是 attach)

离开当前会话
Ctrl+b d

关闭当前会话
exit

在会话里面列出所有会话
Ctrl+b s

重命名会话
tmux rename-session -t 0

当前会话右边加一个窗格
ctrl+b %

划分上下两个窗格
Ctrl+b “

光标切换到不同窗格
trl+b 方向键上下左右

关闭当前窗格
Ctrl+b x

调整窗格大小 (mac 按键冲突了)
Ctrl+b Ctrl+方向键

最大化独立窗口, 再使用一次会变回原来大小
Ctrl+b z

当前窗格与上下一个窗格交换位置
Ctrl+b {
Ctrl+b }

创建一个新窗口，状态栏会显示多个窗口的信息
Ctrl+b c

从列表中选择窗口
Ctrl+b w

tmux 查看历史输出
在会话中

即进入历史输出信息查看模式
可通过键盘上的上下左右键来滚动历史输出信息。
如果要退出查看模式，按下 q 即可

重开机自动恢复tmux
装插件 tmux-continuum

配置 tmux

复制粘贴
mac
从一开始按住 fn
然后 command c v 复制粘贴

linux (还没确认)
从一开始按住 shift
然后 ctrl c v 复制粘贴
